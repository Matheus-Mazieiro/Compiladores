# Compilador para a Linguagem LA
Este projeto é parte do trabalho da disciplina de Compiladores na UFSCar, orientado pelo Prof. Jander. O objetivo é implementar, em etapas, um compilador para a linguagem LA (Linguagem Algorítmica), desenvolvida no âmbito do DC/UFSCar.

- [T1 - Analisador Léxico](https://github.com/Matheus-Mazieiro/Compiladores/tree/master/T1)
- [T2 - Analisador Sintático](https://github.com/Matheus-Mazieiro/Compiladores/tree/master/T2)
- [T3 - Analisador Semântico](https://github.com/Matheus-Mazieiro/Compiladores/tree/master/T3)
- [T4 - Analisador Semântico](https://github.com/Matheus-Mazieiro/Compiladores/tree/master/T4)
- [T5 - Gerador de Código](https://github.com/Matheus-Mazieiro/Compiladores/tree/master/T5)
- [T6 - Trabalho Final](https://github.com/Matheus-Mazieiro/Compiladores/tree/master/T6)
 
### Tecnologias Utilizadas
- [Go (1.22+)](https://go.dev)

- [Antlr](https://www.antlr.org/) — para geração do analisador a partir de uma gramática .g4
